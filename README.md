# regressionProject

Analysis of mtcars data set which is a collection of cars. main interest of
exploration is to understand the relationship between a set of variables and 
miles per gallon (MPG) (outcome).
Main aspects:
1. Is an automatic or manual transmission better for MPG
2. Quantify the MPG difference between automatic and manual transmissions

All covariate model is used for regression purpose and a number of exploratory 
analysis tasks have been performed to check the validity of models.
